## ✨ Features

- 🤖 **Unbeatable AI**: Powered by the Minimax algorithm. Challenge yourself against a machine that never loses.
- 🎨 **Neon Aesthetic**: A deep-space purple and cyan theme with glowing effects and glassmorphism.
- 🏆 **Score Tracking**: Keep tabs on your wins, draws, and the AI's dominance.
- 🎇 **Win Animations**: Dynamic celebratory effects using `canvas-confetti`.
- 📱 **Responsive Design**: Fully playable on desktop and mobile viewports.

## 🚀 Quick Start

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/aishikcodes/Tic-Tac-Toe-game-.git
   cd Tic-Tac-Toe-game-
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

## 🛠️ Built With

- **React 18** - UI Logic & Component Architecture
- **TypeScript** - Type safety and robust development
- **Vite** - Lightning-fast build tool and HMR
- **Vanilla CSS** - Custom neon styling and keyframe animations
- **Canvas-Confetti** - High-performance particle effects

## 🕹️ Game Modes

- **PvE (Player vs Environment)**: Test your skills against the AI.
- **PvP (Player vs Player)**: Challenge a friend locally on the same device.
